# Harmony - Modern Music Website 🎵

---

## 📄 Project Description
**"Harmony"** is a premium, modern web design for a music streaming or music store platform. Featuring a dark mode UI with vibrant neon accents, this project is designed to give users an immersive audio-visual vibe. It showcases advanced front-end design, responsive layout, and structured interactive sections.


---

## 🛠 Technologies Used
* **HTML5**: For semantic page structuring (Home, About, Pricing, Music, Contact).
* **CSS3**: For advanced dark-theme styling, neon glowing effects, custom animations, and responsiveness.
* **JavaScript**: For interactive audio components and UI controls.
* **Git & GitHub**: For professional project management and hosting.
